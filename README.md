### Getting Started (Local Development)

**1. Clone the repository:**
```bash
git clone https://github.com/filipposobrijanu/Gym-Website.git
cd Gym-Website
```

**2. Local Deployment Workflow:**
* This project runs as a zero-dependency static frontend ecosystem. 
* To initialize a local environment, open the `index.html` file natively within your preferred web browser, or serve it using the **Live Server** extension inside VS Code to monitor design manipulations instantly.
